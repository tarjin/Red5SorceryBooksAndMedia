Research

RQSM™ — Red5Sorcery Quantum Security Model

A steganographic system that hides ZIP archives inside paired PNG images — without encryption. No keys to lose. No ciphertext to flag. Just an image.

Built in Rust and compiled to WebAssembly, RQSM™ runs entirely in your browser. Nothing is uploaded to a server. Designed for journalists, activists, and anyone who needs to preserve what matters quietly.

Try CloakYourSecrets.com
// RQSM™ — steganographic embed
fn embed_archive(
  carrier: &PngImage,
  anchor: &PngImage,
  payload: &[u8]
) -> Result<PngImage> {

  // no keys. no ciphertext.
  // just presence and absence.

  let delta = anchor.diff(carrier);
  delta.encode(payload)
}
